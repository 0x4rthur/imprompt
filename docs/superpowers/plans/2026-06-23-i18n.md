# i18n (EN principal + pt-BR) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Tornar o Imprompt bilíngue (inglês padrão + português-BR), com troca de idioma ao vivo, cobrindo janela principal, popup, bandeja/notificações Rust e os 5 presets default.

**Architecture:** Solução custom-leve. Fonte da verdade = `Settings.locale`. Frontend: dicionário tipado + `t()`/`<Trans>` + hook `useT()` (via `useSyncExternalStore`), compartilhado pelas duas janelas React (cada uma lê o locale do backend via `get_settings`). Backend: módulo `i18n.rs` com tabela `(key, en, pt)` para bandeja/notificações/erros; mensagens de erro viram chaves traduzidas na borda. Presets: `default_presets(locale)`.

**Tech Stack:** Tauri 2 (Rust), React 18 + TypeScript + Vite, Vitest (a adicionar p/ teste de paridade do front).

**Spec:** `docs/superpowers/specs/2026-06-23-i18n-design.md`

- [ ] **Step 2: Store do locale + `t()`**

`src/i18n/index.ts`:

```ts
import { CATALOG, type Key, type Locale } from "./catalog";

let current: Locale = "en";
const listeners = new Set<() => void>();

export function getLocale(): Locale {
  return current;
}
export function setLocale(l: Locale): void {
  if (l !== current && l in CATALOG) {
    current = l;
    listeners.forEach((fn) => fn());
  }
}
export function subscribe(fn: () => void): () => void {
  listeners.add(fn);
  return () => listeners.delete(fn);
}
export function lookup(l: Locale, key: Key): string {
  const table = CATALOG[l] as Record<string, string>;
  return table[key] ?? (CATALOG.en as Record<string, string>)[key] ?? key;
}
export function t(key: Key, params?: Record<string, string | number>): string {
  let s = lookup(current, key);
  if (params) {
    for (const [k, v] of Object.entries(params)) {
      s = s.split(`{${k}}`).join(String(v));
    }
  }
  return s;
}
```

- [ ] **Step 4: `<Trans>` para strings com markup inline**

`src/i18n/Trans.tsx`:

```tsx
import { Fragment, type ReactNode } from "react";
import { getLocale, lookup } from "./index";
import type { Key } from "./catalog";

// Interpola nós React em placeholders {nome}. Ex.:
// <Trans k="gatilho.help" slots={{ kbd: <kbd>Ctrl+C</kbd> }} />
export function Trans({ k, slots }: { k: Key; slots: Record<string, ReactNode> }) {
  const s = lookup(getLocale(), k);
  const parts = s.split(/(\{[a-zA-Z0-9_]+\})/g);
  return (
    <>
      {parts.map((p, i) => {
        const m = p.match(/^\{([a-zA-Z0-9_]+)\}$/);
        return <Fragment key={i}>{m ? (slots[m[1]] ?? p) : p}</Fragment>;
      })}
    </>
  );
}
```

> `Trans` lê `getLocale()` direto; quem renderiza deve estar sob um componente que usa `useT()` (re-render no toggle). Onde `Trans` for usado fora de um consumidor de `useT`, garantir que o ancestral chame `useT()` (App e cada Tab já chamarão).

`src-tauri/src/i18n.rs`:

```rust
//! i18n.rs — strings de chrome do BACKEND (bandeja, notificações, erros).
//! Cada entrada carrega EN e PT → paridade por construção. Fallback: EN, depois a própria chave.

/// (key, en, pt-BR)
const ENTRIES: &[(&str, &str, &str)] = &[
    // tray
    ("tray.prefs", "Preferences", "Preferências"),
    ("tray.undo", "↩ Undo last imprompt", "↩ Desfazer último imprompt"),
    ("tray.update", "Check for updates", "Verificar atualizações"),
    ("tray.quit", "Quit", "Sair"),
    // notif (exemplos — completar na Task 6)
    ("notif.update.available", "Update {0} available.", "Atualização {0} disponível."),
    ("notif.update.uptodate", "You're on the latest version.", "Você já está na última versão."),
];

fn entry(key: &str) -> Option<&'static (&'static str, &'static str, &'static str)> {
    ENTRIES.iter().find(|e| e.0 == key)
}

pub fn tr(locale: &str, key: &str) -> &'static str {
    match entry(key) {
        Some((_, en, pt)) => if locale == "pt-BR" { pt } else { en },
        None => "", // chamado só com chaves conhecidas (tray/notif); typo → vazio (pego no teste manual)
    }
}

/// Traduz se `msg` for uma chave CONHECIDA; senão devolve a própria msg (detalhe técnico cru).
/// Usado na borda de erro, onde o texto pode ser uma chave OU um erro técnico — sem leak.
pub fn tr_msg(locale: &str, msg: &str) -> String {
    if entry(msg).is_some() {
        tr(locale, msg).to_string()
    } else {
        msg.to_string()
    }
}

pub fn tr_args(locale: &str, key: &str, args: &[&str]) -> String {
    let mut s = tr(locale, key).to_string();
    for (i, a) in args.iter().enumerate() {
        s = s.replace(&format!("{{{i}}}"), a);
    }
    s
}
```
